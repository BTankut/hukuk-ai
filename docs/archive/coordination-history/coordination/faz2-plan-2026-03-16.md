## 2. EVAL METHODOLOGY

- **Inner-loop:** 1 run veya kategori-slice smoke eval

## 3. PRIORITY WAVES

### Wave 1 — Eval Güvenilirliği + Hata Taksonomisi ✅ TAMAMLANDI
- 170 soru (test_questions_v3.json), commit `4caad91`
- Baseline eval çalışıyor

### Wave 2 — Prompt Mimarisi + Structured Reasoning + Norm Seçimi

**Amaç:** Modelin yanlış kanun/maddeye gitmesini azalt.

**Yaklaşım:**
1. Law-family selection step (zorunlu ara adım)
2. Structured hidden reasoning (CoT: issue_type → law_family → candidate_articles → chosen_article → evidence_span → answer)
3. Evidence-first answer policy
4. Citation discipline rules
5. Few-shot + contrastive examples (TMK/TBK karışan örnekler)
6. Cross-law disambiguation
7. Answer compactness guardrail
8. Tetikleyicili query rewrite (ayrı wave değil, Wave 2 içinde)

**Erken Trigger (tmk_cross_law):**
- 1 günlük iterasyon sonrası src artışı <+5pp → Wave 5 veri hazırlığı başlar

**Kabul Kriteri (3-run):** global src ≥+2.5pp, tmk_cross_law src ≥+5pp, cit ≥+2pp, refusal <1pp düşüş
**Efor:** 3-4 gün

### Wave 3 — Selective Re-Ranking + Evidence Compression ‖ PARALEL

**Amaç:** Retrieved context'te doğru maddeyi ön plana çıkar.

**Yaklaşım:**
- Cross-encoder reranker (birden fazla law-family top-k'da geldiğinde)
- Compressed top-3 evidence pack
- Latency artışı ≤15%

**Kabul Kriteri (3-run):** tmk_cross_law src ≥+5pp veya wrong_law hata ≥30% azalma, global src ≥+1.5pp
**Efor:** 2-3 gün

### Wave 4 — Selective Verification Engine ‖ PARALEL

**Amaç:** Citation-claim tutarsızlığını yakala.

**Yaklaşım:**
- Selective trigger (cross-law, yakın maddeler, tutarsızlık)
- Lightweight consistency check → tutarsızlıkta regeneration
- Trigger rate ≤25%

**Kabul Kriteri (3-run):** hal ≤6% veya ≥-1.5pp, cit ≥+2pp, refusal ≤1pp düşüş
**Efor:** 3-5 gün

### Wave 5 — Hedefli LoRA Fine-Tuning (Conditional, Early Trigger)

**Tetiklenme:**
- Erken: Wave 2 sonrası tmk_cross_law <+5pp
- Geç: Wave 3-4 plato

**Kapsam:** TMK/TBK disambiguation, wrong_article correction, citation discipline
**Efor:** 4-6 gün (pilot + eval)
