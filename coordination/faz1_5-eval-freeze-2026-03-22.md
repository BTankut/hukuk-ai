## Frozen Expectations

The FAZ 1.5 comparison will be considered technically matched only if:

- dataset file remains the same frozen path and hash
- runner family remains `eval_runner`
- baseline and candidate reports are generated by the same metric code hash
- `report_meta.eval_family` matches the family under test
- `report_meta.model_ref` and `report_meta.checkpoint_ref` are explicit
- outputs are published under `evaluation/reports/`

## Caveat

This freeze artefact only freezes the comparison ground. It does not itself grant cutover readiness; that depends on the matched report results, taxonomy, readiness matrix, topology rehearsal, and final decision package.
